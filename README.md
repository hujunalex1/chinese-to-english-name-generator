# 中文名转英文名生成器

这是一个帮助中国人生成有趣且富有文化内涵英文名的网站。

## 功能特点
- 输入中文名，生成三个独特的英文名
- 每个英文名都融入中国文化元素
- 提供中英文的名字寓意解释
- 简洁直观的用户界面
- 响应式设计，支持各种设备

## 技术栈
- HTML5
- CSS3 (Flexbox布局)
- JavaScript (原生)
- DeepSeek V3 API

## 文件结构
```
name_format/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # JavaScript逻辑
└── README.md       # 项目说明
```

## API说明
本项目使用DeepSeek V3 API生成英文名，通过API实现：
- 基于中文名的语义理解
- 生成具有中国文化特色的英文名
- 提供详细的名字寓意解释
